Qobuz authentication

Your credentials are stored in the container's config volume and never leave your server. To find your auth token: sign in at play.qobuz.com, open dev tools (F12) → Application → Local Storage → play.qobuz.com, and copy the localuser entry's token.

{% if creds_from_env %}
Credentials are being provided by the QOBUZ_USER_AUTH_TOKEN environment variable (set in compose.yaml or .env). Anything you save here is written to the streamrip config file but will be overridden on the next container restart. Unset the env var to manage credentials from this page.
{% endif %}

Mode
