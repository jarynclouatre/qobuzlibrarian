Qobuz authentication

Paste your Qobuz user token below. Qobuz has no public login API, so the app reuses the token your own browser already holds — stored only on your server, never sent anywhere but Qobuz.

How do I find my token?

Sign in at play.qobuz.com, open dev tools (F12), then go to Local Storage → play.qobuz.com (Chrome/Edge: Application tab; Firefox: Storage tab). Click the localuser entry and copy its token value — the id in the same entry is your user ID.

{% if creds_from_env %}
Credentials are being provided by the QOBUZ_USER_AUTH_TOKEN environment variable (set in compose.yaml or .env). Anything you save here is written to the streamrip config file but will be overridden on the next container restart. Unset the env var to manage credentials from this page.
{% endif %}
