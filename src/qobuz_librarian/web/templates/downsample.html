Downsample to CD rate

Find FLACs in your library stored above CD rate (88.2 / 96 / 176.4 / 192 / 352.8 / 384 kHz) and shrink them to 44.1 / 48 kHz to reclaim space — most playback gear can't tell the difference. You review the list and tick the albums to shrink; nothing is selected by default.

This can't be undone. Each file is rewritten in place and the hi-res original is not kept — re-downloading hi-res later would just reverse the saving. Every file is verified to decode cleanly before it replaces the original.

Local only — no Qobuz login needed. Only FLAC files are touched; lossy files and files already at CD rate are left alone.
